# Додаток для реєстрації та входу користувачів

## Огляд

Цей додаток призначений можливості реєстрації, входу користувачів.

## Ролі користувачів і доступ до функціональності

У нашому додатку реалізована система ролей користувачів, яка визначає їхні права доступу та функціональність. Нижче наведено перелік ролей та пов'язані з ними можливості:

### Роль `ROLE_ADMIN`

Користувачі з роллю `ROLE_ADMIN` мають доступ до адміністративних функцій, таких як:
- Управління користувачами: додавання, видалення, редагування облікових записів інших користувачів.
- Налаштування системи: зміна параметрів додатку, конфігурація прав доступу.

### Роль `ROLE_USER`

Користувачі з роллю `ROLE_USER` мають доступ до основних функцій додатку, а саме:
- Перегляд інформації: доступ до основних функцій, таких як перегляд даних, списків, звітів.
- Додавання та редагування: можливість додавати нові дані, редагувати існуючі записи, відповідно до їхніх прав доступу.

## Контролери

### Пакет `package ua.ypon.accounting.controllers;`

#### Реєстрація та вхід

- **RegistrationController**: Контролер для обробки реєстрації нових користувачів та входу в систему.
    - `/hello`: Показує привітальну сторінку.
    - `/login`: Показує сторінку входу.
    - `/registration`: Показує сторінку реєстрації та обробляє дані форми реєстрації.

# Додаток для обліку особистих витрат

## Огляд

Цей додаток призначений для ведення обліку особистих витрат. Він забезпечує можливість записування їхніх витрат з різних категорій.

## Контролери

### Пакет `ua.ypon.accounting.controllers.personal`

#### Особисті витрати

- **PersonalExpensesController**: Контролер для управління особистими витратами.
    - `/api/expenses`: Повертає всі особисті витрати у форматі JSON.
    - `/new`: Показує форму для створення нової витрати.
    - `/add_expenses`: Обробляє дані форми для додавання нових витрат.
    - `/show`: Показує всі збережені витрати.
    - `/edit/{id}`: Показує форму для редагування конкретної витрати.
    - `/update/{id}`: Обробляє дані форми для оновлення витрати.
    - `/delete/{id}`: Видаляє конкретну витрату.

#### Інші типи особистих витрат

- **FoodExpensesController**: Контролер для управління витратами на харчування.
    - `/api/sum_food_expenses`: Повертає загальну суму витрат на харчування.
    - `/api/sum_expenses_food_between_date`: Повертає суму витрат на харчування за певний період.

- **OtherExpensesController**: Контролер для управління іншими типами витрат.
    - `/api/sum-other-expenses`: Повертає загальну суму інших витрат.
    - `/api/sum_expenses_other_between_date`: Повертає суму інших витрат за певний період.

- **UtilityExpensesController**: Контролер для управління витратами на комунальні послуги.
    - `/api/sum-utility-expenses`: Повертає загальну суму витрат на комунальні послуги.
    - `/api/sum_expenses_utility_between_date`: Повертає суму витрат на комунальні послуги за певний період.

#### Оновлення витрат у партіях

- **BatchUpdateController**: Контролер для пакетного оновлення особистих витрат.
  - `/new_batch`: Показує форму для пакетного оновлення особистих витрат.
  - `/process_batch`: Обробляє дані форми для оновлення особистих витрат у партіях.
  - `/add_expenses`: Обробляє дані форми для додавання нових особистих витрат у партіях.

- **IncomeController**: Контролер для управління доходами від магазинів.

#### Інші типи доходів

  - `/api/total_sum_other_between_date`: Повертає суму інших доходів за певний період.

Цей додаток призначений для ведення обліку та управління закупівлями та витратами. Він забезпечує можливість реєстрації, оновлення та перегляду різних видів закупівель та пов'язаних із ними витрат.

  - `/api/sum_expenses_purchasing_owner_between_date`: Повертає суму витрат від власника за певний період.

#### Управління закупівлями від постачальників

- **SuppliersPurchasingController**: Контролер для управління закупівлями від постачальників.
  - `/api/total_sum_purchasing_suppliers`: Повертає загальну суму закупівель від постачальників.
  - `/api/sum_expenses_purchasing_suppliers_between_date`: Повертає суму витрат від постачальників за певний період.

### Пакет `ua.ypon.accounting.controllers.business`

#### Управління бізнес-витратами

- **BusinessExpensesController**: Контролер для управління бізнес-витратами.
  - `/api/business_expenses`: Повертає всі бізнес-витрати.
  - `/newBusinessExpense`: Показує форму для створення нової бізнес-витрати.
  - `/show`: Показує сторінку з усіма бізнес-витратами.
  - `/index`: Показує сторінку з усіма бізнес-витратами.
  - `/{id}`: Показує деталі конкретної бізнес-витрати.
  - `/{id}/edit`: Показує форму для редагування конкретної бізнес-витрати.
  - `/{id}` (DELETE): Видаляє конкретну бізнес-витрату.

#### Управління автомобільними витратами

- **CarExpensesController**: Контролер для управління автомобільними витратами.
  - `/api/total_sum_fuel`: Повертає загальну суму витрат на паливо.
  - `/api/total_sum_maintenance`: Повертає загальну суму витрат на технічне обслуговування.
  - `/api/sum_expenses_fuel_between_date`: Повертає суму витрат на паливо за певний період.
  - `/api/sum_expenses_maintenance_between_date`: Повертає суму витрат на технічне обслуговування за певний період.

#### Управління операційними витратами

- **OperationExpensesController**: Контролер для управління операційними витратами.
  - `/api/total_sum_salary_v`: Повертає загальну суму витрат на заробітну плату (персонал_V).
  - `/api/total_sum_salary_i`: Повертає загальну суму витрат на заробітну плату (персонал_I).
  - `/api/total_sum_utility`: Повертає загальну суму витрат на комунальні послуги.
  - `/api/total_sum_rent`: Повертає загальну суму витрат на оренду.
  - `/api/total_sum_tax_single`: Повертає загальну суму одноразового податку.
  - `/api/total_sum_tax_pension`: Повертає загальну суму внесків до пенсійного фонду.
  - `/api/sum_expenses_salary_v_between_date`: Повертає суму витрат на заробітну плату (персонал_V) за певний період.
  - `/api/sum_expenses_salary_i_between_date`: Повертає суму витрат на заробітну плату (персонал_I) за певний період.

#### Пакетне оновлення бізнес-витрат

- **BatchUpdateBusinessController**: Контролер для пакетного оновлення бізнес-витрат.
  - `/new_batch`: Показує форму для пакетного оновлення бізнес-витрат.
  - `/process_batch`: Обробляє дані форми для оновлення бізнес-витрат.
  - `/add_expenses`: Обробляє дані форми для додавання нових бізнес-витрат.

#### Збірка білда та запуск контейнера в Docker

- **Dockerfile**: Файл для створення Docker-образу додатку.
  - `docker build -t accounting .`: Команда для створення Docker-образу.
  - `docker run -rm -p 8080:8080 accounting`: Команда для запуску контейнера з додатком.
